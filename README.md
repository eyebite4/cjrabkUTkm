# Python招聘数据爬取分析可视化系统

## 核心功能
- 数据爬取：利用Python技术爬取BOSS直聘网站上的岗位信息。
- 模拟登陆：模拟BOSS直聘网站用户登陆，获取爬取权限。
- 数据提取：使用beautifulsoup从网页源码中提取岗位信息。
- 数据存储：将提取的岗位信息存储到MySQL数据库中。
- 数据可视化：使用pyecharts生成柱状图或饼状图等可视化图表。
- 趋势分析：通过可视化分析，对比分析职位的趋势。

## 适用场景
- 人力资源部门：用于分析市场招聘趋势，优化招聘策略。
- 企业招聘人员：用于了解行业招聘趋势，制定招聘计划。
- 行业分析师：用于分析行业招聘情况，提供行业报告。

## 技术特点
- Python语言：使用Python进行编程，提高开发效率。
- Django后端框架：使用Django构建后端系统，确保系统稳定性。
- MySQL数据库：使用MySQL存储数据，保证数据安全。
- pyecharts可视化：使用pyecharts进行数据可视化，提升数据展示效果。
- selenium爬虫：使用selenium进行网页自动化，提高数据爬取效率。
- 词云：通过词云展示热门关键词，快速了解岗位需求。
